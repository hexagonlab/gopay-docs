3rd party апп-аас тухай мерчант-ын invoice-ыг дүн болон бусад мэдээллийг QR image-с авсан кодоор лавлаж авна.

## QR үүсгэх хүсэлт

| METHOD | URL                            | HEADER                                                           |
| ------ | ------------------------------ | ---------------------------------------------------------------- |
| POST   | `/functions/v1/get-qr-invoice` | `Content-Type: application/json` `Authorization: Bearer {Token}` |

### Request body (JSON)

| Параметер | Төрөл | Тайлбар      |
| --------- | ----- | ------------ |
| code      | uuid  | invoice code |

### Request response

| Нэр     | Төрөл   | Тайлбар                                               |
| ------- | ------- | ----------------------------------------------------- |
| success | boolean | амжилттай эсэх                                        |
| invoice | Invoice | Үүссэн invoice-ын дэлгэрэнгүй мэдээлэл                |
| message | string  | Хэрвээ алдаа гарсан тохиолдолд алдааны тайлбар байна. |

### Invoice model

| Нэр    | Төрөл  | Тайлбар           |
| ------ | ------ | ----------------- |
| code   | uuid   | invoice-ын дугаар |
| amount | double | үүсгэсэн дүн      |
