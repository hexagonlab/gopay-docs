---
title: Зээлийн баталгаажуулалт хийх
description: QR Invoice-ын дагуу зээл үүссэнийг баталгаажуулах
---

3rd party апп-аас тухай мерчант-ын invoice-ыг код болон бусад мэдээллээр зээлийн баталгаажуулалт хийснээр мерчантад зээл амжилттай болж төлбөр шилжилсэн талаар мэдэгдэл очно.

## QR үүсгэх хүсэлт

| METHOD | URL                            | HEADER                                                           |
| ------ | ------------------------------ | ---------------------------------------------------------------- |
| POST   | `/functions/v1/pay-qr-invoice` | `Content-Type: application/json` `Authorization: Bearer {Token}` |

### Request body (JSON)

| Параметер        | Төрөл  | Тайлбар            |
| ---------------- | ------ | ------------------ |
| code             | uuid   | invoice code       |
| issuer_lender_id | long   | issuer lender ID   |
| first_name       | string | харилцагийн нэр    |
| last_name        | string | харилцагийн овог   |
| id_number        | string | регистерийн дугаар |
| phone_number     | string | утасны дугаар      |

### Request response

| Нэр     | Төрөл   | Тайлбар                                               |
| ------- | ------- | ----------------------------------------------------- |
| success | boolean | амжилттай эсэх                                        |
| data    | mixed   | Зээл үүссэнийг хүлээн авсан дэлгэрэнгүй мэдээлэл      |
| message | string  | Хэрвээ алдаа гарсан тохиолдолд алдааны тайлбар байна. |
