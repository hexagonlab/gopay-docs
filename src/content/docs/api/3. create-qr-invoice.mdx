---
title: QR Invoice үүсгэх
description: QR invoice үүсгэх
---

3rd party апп-аас тухай мерчант-ын invoice-ыг дүн болон бусад мэдээллийг илгээж үүсгэнэ.

## QR үүсгэх хүсэлт

| METHOD | URL                               | HEADER                                                           |
| ------ | --------------------------------- | ---------------------------------------------------------------- |
| POST   | `/functions/v1/create-qr-invoice` | `Content-Type: application/json` `Authorization: Bearer {Token}` |

### Request body (JSON)

| Параметер          | Төрөл  | Тайлбар          |
| ------------------ | ------ | ---------------- |
| amount             | double | Мөнгөн дүн       |
| merchant_branch_id | long   | Салбарын дугаар  |
| product_id         | long   | product дугаар   |
| lender_id          | long   | Lender-ын дугаар |

### Request response

| Нэр     | Төрөл   | Тайлбар                                               |
| ------- | ------- | ----------------------------------------------------- |
| success | boolean | амжилттай эсэх                                        |
| message | string  | Хэрвээ алдаа гарсан тохиолдолд алдааны тайлбар байна. |
| invoice | Invoice | Үүссэн invoice-ын дэлгэрэнгүй мэдээлэл                |
| qr_data | string  | base64 QR image                                       |

### Invoice model

| Нэр    | Төрөл  | Тайлбар           |
| ------ | ------ | ----------------- |
| code   | uuid   | invoice-ын дугаар |
| amount | double | үүсгэсэн дүн      |
